## Notes

Thanks to the pygame and numpy communities for making cross-platform game development accessible.

## Controls

| Button | Function |
|--------|----------|
| D-pad | Navigate, tune waterfall, adjust WPM |
| A | Confirm, straight key |
| B/X/Y | Answer choices (Koch trainer) |
| R1 | Play/hear audio |
| R2 | Adjust noise, clear text |
| Select | Back to menu |
| Select+Start | Quit |

## Features

- Straight key practice with sidetone
- Koch method receive trainer with spaced repetition
- Simulated HF waterfall band explorer with realistic noise
- CW glossary with Q-codes and abbreviations
- Callsign challenges (CQ, signal reports, 73)

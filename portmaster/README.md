Thanks to the pygame and numpy communities for making cross-platform game development accessible.

## Controls

| Button | Function |
|--------|----------|
| D-pad | Navigate menus, tune waterfall, adjust WPM |
| A | Confirm, straight key, start round |
| B/X/Y | Answer choices (Koch trainer) |
| L1/R1 | Iambic paddle (dit/dah), play/hear audio |
| L2 | Take screenshot |
| Select | Back to menu |
| Select+Start | Quit |

## Features

- Straight key practice with real-time decode and sidetone
- Iambic keyer (Mode A/B) via shoulder buttons or 3.5mm paddle mod
- Koch method receive trainer with spaced repetition
- Vocabulary quiz on CW abbreviations and Q-codes
- QSO procedure trainer with scripted exchanges
- Simulated HF waterfall band explorer with realistic noise
- CW glossary with Q-codes, prosigns, and abbreviations
- Multiple user profiles with independent progress
